## Meeting Document Format

Each meeting document includes:

1. **Transcript** - Speaker-tagged transcription (Thai/English)
2. **Meeting Notes** - Structured technical notes
3. **Technical Decisions** - Architecture and design decisions
4. **Action Items** - Tasks with assignees and priorities
5. **Risks/Issues** - Technical concerns and mitigation plans
6. **Dependencies** - Blockers and pending items
7. **Executive Summary** - Bilingual summaries (Thai and English)
